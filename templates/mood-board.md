# 情绪板模板

## 使用说明

美术方向确认用。展示色彩基调、材质参考、灯光氛围和纹理方向。

## Prompt 模板

```text
电影级情绪板 / 视觉氛围参考，主题《[片名]》。16:9 横版，professional mood board / visual atmosphere reference board。包含模块：(1) 色彩色卡 — [5-8] 个色块，标注 [色名/用途/十六进制感]，配色方案 [配色方案名]；(2) 材质贴图参考 — [材质1]、[材质2]、[材质3]、[材质4] 的局部纹理展示；(3) 灯光氛围小图 — [3-4] 个小型灯光效果展示：[灯光方案1]、[灯光方案2]、[灯光方案3]；(4) 字体参考 — [标题字体风格] + [正文字体风格]，作为质感展示；(5) 纹理参考 — [纹理1/纹理2/纹理3] 的环境应用效果。整体色调 [主色1] + [主色2] + [点缀色]，氛围 [氛围描述]。风格方向：[风格编号. 风格名称]。professional mood board, color palette swatches with labels, material texture samples, lighting atmosphere references, typography samples, cohesive visual theme, cinematic color grading, ultra-detailed, 8K, clean layout, no watermark, no garbled text, no random elements, organized and labeled sections。
```
