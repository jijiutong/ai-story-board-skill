# 角色设定卡模板

## 使用说明

角色设计确认用。包含两种模式：标准角色设定卡（格式 3）和三视图设定卡（格式 9）。

## 模式 A：标准角色设定卡（格式 3）

```text
电影级角色设定卡，主题《[片名]》，[主角/反派]：[角色姓名]。[画幅比例，默认4:3] 横版。角色设定展示：[主角姓名/身份]，年龄 [年龄]，[服装描述]，[面部特征]，[体型特征]，[情绪底色]。布局包含：正面全身像、侧面全身像、背面全身像（三视图严格对齐），面部 close-up（表情 [表情描述]），核心道具/武器特写 [道具描述]，服装细节展示 [面料/配饰/纹理]。角色造型风格：[风格编号. 风格名称]，[氛围关键词]，配色 [主色1] + [主色2] + [点缀色]。character turnaround sheet, front/side/back aligned views, detailed facial close-up, prop close-up, costume details, consistent character proportions, professional character design sheet, ultra-detailed, 8K, sharp focus, no watermark, no garbled text, no broken faces, no extra limbs, no duplicated limbs, consistent style throughout。
```

## 模式 B：三视图设定卡（格式 9）

```text
专业三视图设定卡，主题《[片名]》，角色：[角色姓名]。16:9 横版，严格正交三视图布局：正面全身、侧面全身、背面全身，三个视图严格对齐同一水平线，相同比例，无透视变形。标注：身高 [身高描述]、各部位尺寸比例、材质标注 [材质1/材质2/材质3]、颜色编号 [颜色描述]。背景纯白或中性灰。角色造型：[风格编号. 风格名称]，[服装描述]。technical character turnaround, orthographic front/side/back views, aligned on same baseline, same scale, no perspective distortion, dimension annotations, material callouts, color swatches, clean white background, professional 3D modeling reference, ultra-detailed, 8K, sharp focus, no watermark, no background elements, no shadows。
```
