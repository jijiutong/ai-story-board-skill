# 快速故事板模板

## 使用说明

精简版故事板，适合快速验证想法和单场景分镜参考。比全案板少了角色三视图和技术规范栏。

## Prompt 模板

```text
电影级漫剧故事板，主题《[片名]》，核心冲突：[一句话故事]。[画幅比例，默认16:9] 横版，专业电影前期制作分镜板，顶部片名和信息栏，中部超大核心场景概念图展示 [场景地点] + [天气] + [氛围]，[场景地点] 的对峙画面，底部 [7/13] 镜分镜表 + 简易镜头轨迹图。风格：[风格编号. 风格名称]，[氛围关键词]，配色 [主色1] + [主色2] + [点缀色]，cinematic storyboard sheet, professional film production layout, ARRI ALEXA 35, anamorphic lens, strong contrast lighting, film grain, volumetric light, ultra-detailed, 8K, clean industrial layout, coherent character design, no watermark, no garbled text, no messy panels, no broken anatomy, no flat illustration, no marketing poster style。
```

## 关键帧模式（格式 10）

当用户选择"关键帧序列"时，用以下模板：

```text
电影级关键帧序列，主题《[片名]》，[动作/场景描述]。[画幅比例，默认16:9] 横版，[5-9] 个关键动作帧，展示 [动作起始] → [动作发展] → [动作高潮] → [动作收尾] 的完整过程。每个关键帧是小横幅电影帧，标注动作阶段和时间。主角 [主角描述] 在 [场景描述] 中执行 [动作描述]，对手 [对手描述]。风格：[风格编号. 风格名称]，[氛围关键词]，配色 [主色1] + [主色2] + [点缀色]，动作轨迹线标注，特效标注，consistent character across all frames, motion blur on moving elements, cinematic key frame sequence, ultra-detailed, 8K, sharp focus, no watermark, no garbled text, no broken faces, no extra limbs。
```
