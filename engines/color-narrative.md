# 色彩叙事引擎

根据情绪弧线动态调整色彩。数据表见 `knowledge/color-narratives.md`。

---

## 决策逻辑

### 步骤1：识别情绪弧线

从情绪曲线编号映射色彩叙事编号：

| 情绪曲线 | 推荐色彩方案 |
|---------|-------------|
| EC1 标准四阶段 | CN4 愤怒爆发 / CN5 治愈温暖 |
| EC2 悬疑五阶段 | CN3 悬疑逼近 |
| EC3 喜剧四阶段 | CN16 青春明亮 |
| EC4 悲剧四阶段 | CN2 虐心刀子 / CN6 孤独终局 |
| EC5 爱情五阶段 | CN1 爱情升温 / CN41 浪漫梦幻 |
| EC6 成长六阶段 | CN7 希望升起 |
| EC7 复仇五阶段 | CN8 堕入黑暗 / CN4 愤怒爆发 |

> 完整50+色彩方案见 `knowledge/color-narratives.md`

---

### 步骤2：获取色彩参数

根据色彩方案编号，从 `knowledge/color-narratives.md` 获取：
- 四阶段色彩演进
- 主色/辅色/点缀色/暗部
- 适用类型

---

### 步骤3：联动情绪曲线

色彩随情绪曲线阶段动态变化：
- 阶段1 平静 → 对应色彩方案阶段1
- 阶段2 对峙 → 对应色彩方案阶段2
- 阶段3 爆发 → 对应色彩方案阶段3
- 阶段4 余韵 → 对应色彩方案阶段4

---

## Prompt 输出模板

```
色彩叙事：CN[编号]. [方案名称]。
阶段1 平静：[色彩描述]
阶段2 对峙：[色彩描述]
阶段3 爆发：[色彩描述]
阶段4 余韵：[色彩描述]
```

---

## 联动流程图

```
情绪曲线 ──→ color-narrative.md ──→ 色彩方案编号(CN1-CN50+)
                │
                ▼
        knowledge/color-narratives.md ──→ 色彩参数
                │
                ▼
        阶段映射 ──→ 每阶段色彩
```
