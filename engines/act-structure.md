# 叙事结构引擎

根据故事类型自动识别叙事结构。数据表见 `knowledge/narrative-structures.md`。

---

## 决策逻辑

### 步骤1：识别叙事结构

从故事类型映射叙事结构编号：

| 故事类型 | 叙事结构编号 |
|---------|-------------|
| 动作/爱情/科幻/喜剧 | NS1 三幕标准 |
| 爱情/成长/都市/古装 | NS2 四幕结构 |
| 悬疑/犯罪/惊悚 | NS3 倒叙结构 |
| 身份对立/替身/回忆 | NS4 双线叙事 |
| 哲学/文艺/悲剧 | NS5 环形叙事 |
| 悬疑/人性/群像 | NS6 多视角叙事 |

> 完整12种叙事结构见 `knowledge/narrative-structures.md`

---

### 步骤2：获取结构参数

根据结构编号，从 `knowledge/narrative-structures.md` 获取：
- 各幕/阶段比例
- 每幕内容要点
- 镜头分配
- 适用类型

---

### 步骤3：映射镜头分配

根据叙事结构分配镜头：
- 三幕结构(7镜)：镜1-2建制 → 镜3-5对抗 → 镜6-7解决
- 四幕结构(7镜)：镜1-2起 → 镜3-4承 → 镜5-6转 → 镜7合

---

## Prompt 输出模板

```
叙事结构：NS[编号]. [结构名称]。
[各幕/阶段描述]
```
