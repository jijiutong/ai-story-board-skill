# AI Visual Director Skill 回归测试集

> 版本：v2.0 · 14 条核心规则
> 测试范围：题材泛化 / 分镜稳定性 / 一致性 / 风格迁移 / Prompt 可执行性

---

## 1. 测试目标

本测试集用于验证 AI Visual Director Skill 在以下维度的稳定性：

1. **题材识别** — 不同题材是否匹配正确的推荐风格
2. **风格推荐** — 推荐是否使用通用名、不自动融合、不暴露内部编号
3. **/storyboard 一键生成** — 7 模块是否完整输出、角色/场景锚点是否到位
4. **/character 角色锚点** — 剧情身份标注、身份冲突解释、无文字标注
5. **/scene 场景锚点** — 出图模式默认无标注、光源/空间/氛围完整
6. **单镜修改** — 保留镜号/时间码/顺序、天气变更自动补桥
7. **风格迁移** — 声明锁定不变项、不改剧情事实、古代场景不出现现代光源词
8. **题材禁忌** — 不串味、不过度史诗化、不出图冲突、不古今穿帮

---

## 2. 评分标准

每条测试用例按 10 个维度评分，每项 1-10 分：

| # | 维度 | 满分标准 |
|---|------|---------|
| 1 | 题材识别 | 关键词匹配正确，推荐风格与题材一致 |
| 2 | 风格推荐 | 使用通用风格名，不暴露内部编号，不自动融合 |
| 3 | 情绪强度 | 情绪曲线与故事张力匹配，不夸张不冷淡 |
| 4 | 角色锚点 | 含剧情身份标注，含简要DNA，无红字标注 |
| 5 | 场景锚点 | 含空间+主光源+氛围，出图模式无标注 |
| 6 | 分镜节奏 | 帧数与时长匹配，景别递进合理 |
| 7 | 出图可执行性 | 无文字标注/可读文字，运镜可物理执行 |
| 8 | 连续性 | 修改保留时间码，天气变更自动补桥 |
| 9 | 风格迁移 | 声明显性/锁定不变项，不出现禁用词 |
| 10 | 题材禁忌 | 无串味/无过度史诗化/无古今穿帮 |

**综合评分**：低于 8 分视为需要补丁。
