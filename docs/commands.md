## 6 个主命令

### `/storyboard` — 故事板生成

| 子指令 | 效果 |
|--------|------|
| `/storyboard [故事]` | 粘贴故事 → 智能推荐 2-3 个风格方向 |
| `/storyboard 一键生成 [故事] [时长] [镜数]` | 直接出 7 模块完整故事板 |
| `/storyboard 一键生成 简洁` | 精简输出：角色+场景+分镜表+Prompt |
| `/storyboard 多版本` | A/B/C 三版本对比 |
| `/storyboard 一键全平台` | GPT Image + Midjourney + SD |
| `/storyboard 看全部` | 展示 53 种风格 + 10 种格式 |

**参数**：时长(m:s) / 镜数 / 风格名 / 格式名

### `/character` — 角色设计

| 子指令 | 效果 |
|--------|------|
| `/character [角色名] [描述]` | 默认 6 模块角色设定卡（出图模式，无文字标注） |
| `/character 导演标注版 [角色名] [描述]` | 角色卡 + 红色标注/编号（团队协作） |
| `/character 三视图` | 正/侧/背三视图 |
| `/character 面部多角度` | 5 角度面部特写 |
| `/character 12表情` | 3×4 宫格情绪范围图 |
| `/character 服装武器细节卡` | 服装+武器独立细节 |
| `/character 14图参考` | Nano Banana 最强锚定 |
| `/character IP-Adapter` | ComfyUI 本地锁定 |
| `/character 角色DNA` | 20 字段文字锚定（零成本） |

### `/scene` — 场景设计

| 子指令 | 效果 |
|--------|------|
| `/scene [描述]` | 默认全能参考图（出图模式，无标注） |
| `/scene 导演标注版 [描述]` | 全能参考图 + 箭头/编号/红框 |
| `/scene 720全景` | 360°×180° 全景环境图 |
| `/scene 环绕截图` | 环绕视频→8角度截图 |
| `/scene camera remix` | 同场景多角度镜头变化 |

### `/video` — 视频 Prompt

| 子指令 | 效果 |
|--------|------|
| `出视频 prompt` | 压缩模式（≤1500字），直喂 Seedance |
| `详细模式` / `展开 prompt` | 完整段落（3000+字），适合 Runway/Sora/可灵 |
| `生成视频分镜图` / `合并帧` | 多帧合并为 3-4 张画面锚点 |
| `切分各帧` / `逐帧接力` | 逐帧出图+锁帧链 |
| `生成首尾帧` | 帧1+帧N对比图 |
| `继续下一段` / `续写下一段` | AI 自动续写剧情 + 跨段衔接 |
| `用 [平台] 生成视频` | 调 API 直出（Seedance/Runway/可灵/Luma/Pika） |

### `/style` — 风格调整

| 子指令 | 效果 |
|--------|------|
| `换成 [风格名] 但保持角色` | 风格迁移（只改视觉不改剧情） |
| `多版本` | A/B/C 三版本对比 |
| `压缩到 MJ` / `压缩到 SD` | 平台最佳长度压缩 |
| `看全部` | 浏览 53 种风格 |

### `/poster` — 海报生成

| 子指令 | 效果 |
|--------|------|
| `/poster [故事/描述]` | 电影海报/宣传海报 |

## 全局调整指令

### Mood 滑块

| 指令 | 色彩 | 运镜 | 镜头 |
|------|------|------|------|
| `更燃` | 暖色 | 快切 | 低角度+疾推 |
| `更虐` | 冷色 | 慢镜 | 高角度俯拍+拉远 |
| `更甜` | 柔光+粉 | 环绕 | 浅景深 |
| `更丧` | 去饱和 | 固定 | 空旷构图+空镜 |
| `更暧昧` | 暖暗 | 拉焦 | 负空间+过肩 |
| `更恐怖` | 暗调 | 手持 | 窥视视角+极速推进 |
| `更史诗` | 金色逆光 | 慢推 | 广角+航拍 |
| `更贵` | 材质感+ | — | 摄影参数/色卡/秩序 |

### 单镜修改

```
第X镜 [修改内容]
```

例：`第3镜更暗一点` `第5镜雨改雪` `第7镜更史诗`

约束：保留镜号/时间码/顺序。连续字段（天气/时间/地点/外观）自动补过渡。

### 多画幅

```
适配小红书    → 9:16 竖版
适配抖音      → 9:16
适配朋友圈    → 1:1
适配所有平台  → 自动切换
```

## 工作流

### 快速出片流程

```
粘贴故事 → 一键生成 → 出视频 prompt
```

### 完整视频流程

```
一键生成 → 生成角色卡 → 生成场景参考图 → 生成视频分镜图 → 出视频 prompt
```

### 长视频跨段

```
出下一段 / 继续下一段
用 [技法名] 衔接，继续下一段
```
