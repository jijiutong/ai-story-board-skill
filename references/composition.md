# 构图与版式参考

## 构图法则

| 法则 | 说明 | 适用场景 |
|------|------|----------|
| 三分法 | 主体放在三分线交点 | 通用、自然、和谐 |
| 中心对称 | 主体居中、左右对称 | 庄严、权力、宗教 |
| 引导线 | 利用线条引导视线到主体 | 纵深、路径、河流、道路 |
| 框中框 | 通过门窗/拱门/洞口看主体 | 偷窥感、聚焦、层次 |
| 前景层 | 前景+中景+背景三层深度 | 丰富、电影感、纵深 |
| 对角线 | 主体沿对角线排列 | 动感、紧张、冲突 |
| L型构图 | L形框架内的主体 | 稳定、建筑、城市 |
| 留白 | 大面积空白/暗部+小主体 | 孤独、极简、诗意 |
| 填充画面 | 主体充满整个画面 | 压迫、威胁、力量 |
| 黄金螺旋 | 主体沿斐波那契螺旋排列 | 自然、优雅、经典 |

## 排版规则

- **文字层级**：标题 > 副标题 > 正文 > 注释，字号递减
- **分隔线**：细线（1-2px），金色/银色/白色，不要粗线
- **间距**：模块间留白 > 模块内间距
- **对齐**：左对齐为主，居中对齐用于标题和核心视觉
- **背景**：深色背景（#1a1a1a 或 #0d0d0d）或冷白（#f0f0f0）
- **装饰**：最小化，用线条和色块，不要花纹和图标
