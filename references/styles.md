## 快速引用格式

```
整体风格：[风格编号. 风格名称]，[氛围词]，配色 [配色方案描述]
```

```
整体风格：1. 黑金动作分镜板，黑金硬核、冷灰写实、暗调高反差，配色炭黑+暗金+血红
```
