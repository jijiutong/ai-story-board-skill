# 灯光与色彩参考

## 灯光类型

### 主光（Key Light）
- 场景中最强的光源，决定整体光影方向
- 描述方式：[光源类型] + [方向] + [色温]
- 例：右侧窗户外射入的暖黄色阳光、头顶冷白色荧光灯

### 补光（Fill Light）
- 弱化主光产生的阴影，降低对比
- 描述方式：[光源类型] + [强度描述]
- 例：白色墙壁反射的柔和补光、地面水洼反射的微弱补光

### 轮廓光/逆光（Rim Light / Backlight）
- 从主体后方照射，勾勒边缘轮廓
- 效果：分离主体与背景、增加层次、营造氛围
- 例：强烈的金色逆光勾勒人物剪影边缘

### 环境光（Ambient Light）
- 场景中弥漫的整体光，没有明确方向
- 例：阴天散射光、雾气中的漫射光

### 特效光（Effect Light）
- 特殊效果光源
- 类型：霓虹、全息投影、火焰、电弧、生物发光、魔法光效

## 经典灯光方案

| 方案 | 说明 | 适用 |
|------|------|------|
| 三点布光 | 主光 + 补光 + 轮廓光 | 人物采访、标准拍摄 |
| 逆光剪影 | 强逆光 + 主体欠曝 | 英雄登场、悬念揭示 |
| 伦勃朗光 | 45度主光 + 面部三角光斑 | 人物肖像、深沉情绪 |
| 蝴蝶光 | 正面高位主光 + 鼻下蝴蝶形阴影 | 美女肖像、优雅 |
| 底光/恐怖光 | 从下方照射 | 恐怖、反派、不安 |
| 分割光 | 一侧全亮一侧全暗 | 内心冲突、双面性 |
| 顶光 | 从正上方照射 | 审讯、神圣、压抑 |
| 彩色灯光 | 多色光源混合 | 赛博朋克、夜店、迷幻 |

## 色温参考

| 色温 | 颜色 | 光源 | 情绪 |
|------|------|------|------|
| 1000-2000K | 橙红 | 烛光、篝火 | 温暖、亲密、原始 |
| 2000-3000K | 暖黄 | 钨丝灯、夕阳 | 温馨、怀旧、浪漫 |
| 3000-4000K | 自然白 | 日出日落 | 平和、日常 |
| 4000-5000K | 冷白 | 荧光灯、上午阳光 | 清醒、办公、医院 |
| 5000-6500K | 日光白 | 正午阳光 | 中性、客观 |
| 6500K+ | 冷蓝 | 阴天、月光、屏幕 | 孤独、科技、不安 |

## 配色方案

| 方案 | 说明 | 示例 |
|------|------|------|
| 互补色 | 色环相对两色 | 蓝+橙（电影最常用） |
| 冷暖对比 | 冷色调背景+暖色调主体 | 冷灰街道+暖色人物 |
| 单色系 | 同一色相不同明度 | 不同层次的蓝 |
| 三分色 | 色环上120度间隔三色 | 红+黄+蓝 |
| 相邻色 | 色环上相邻色 | 蓝+绿+青 |
| 黑金 | 黑色底+金色点缀 | 豪华、权力、史诗 |

